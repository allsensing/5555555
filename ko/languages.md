# 다국어

GitBook 은 다국어로 작성된 책 발생을 지원합니다. 각 언어는 일반 GitBook 형식의
하위 디렉토리에 있어야 하며, 다음 형식을의 `LANGS.md` 파일이 저장소의 루트에
존재해야합니다.

```markdown
# 언어

* [English](en/)
* [한국어](ko/)
* [French](fr/)
* [Español](es/)
```

### 각 언어에 대한 구성

언어 책 (예: `en`) 이 `book.json` 을 가지고 있다면, 그 설정은 기본 설정을
확장합니다.

유일한 예외는 플러그인으로 전역에 명시해야하며 언어별로 지정할 수 없습니다.
